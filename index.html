GitHub Test Repo

I'm learning to use GitHub with this test website

Terminal Commands

  1. cd: Change Directory
  2. ls: List files in current directory

GitHub Commands

  1. clone: Clones a repository from GitHub onto local machine
  2. status: Checks the current status of repository
  3. add: Adds the specified file/s to the stage (prepares them for commit)
  4. commit: Commits the changed files
  5. push: Finally uploads committed files to GitHub
