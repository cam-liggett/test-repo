General Tips

  • Type git to show common git commands and general help

Terminal Commands

cd Change directory
ls Lists files and folders of current directory

GitHub Commands

clone Clones a repository onto local machine. Use this the first time you're retreiving a repository.
status Checks the current status of the working branch
add Adds the specified file/s to the stage (prepares them for commit)
commit Commits the changed files to the repository. Always add specific message when committing to describe what you changed (using -m "message")
push Pushes (or uploads) committed files to GitHub repo
pull Pulls (or downloads) repo to local machine. Do this everytime you want to start working and know that changes have been made to repo
